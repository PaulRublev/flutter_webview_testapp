# Sport app / browser

The app shows either a page with the desired URL downloaded remotely, or a stub in the form of a calorie calculator.

---

## Target
Create an application with the following features:
- checking the Internet connection (online/offline)
- requesting a URL string in Firebase Remote Config
- save the URL on the device
- switch to WebView or Calorie Calculator
